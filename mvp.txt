
MVP - Aplicación de Reservas de Canchas de Fútbol

Razón
La aplicación busca resolver el problema de gestión de reservas de canchas de fútbol, 
permitiendo a los usuarios realizar reservas de manera digital, evitando conflictos de 
horarios, optimizando el uso de las instalaciones, gestionando pagos y generando estadisticas.

Impacto
Para usuarios: Facilita el proceso de reserva de canchas sin necesidad de llamadas o visitas presenciales
Para administradores: Mejora la gestión y organización de las reservas
Para el negocio: Reduce costos operativos y mejora la eficiencia

Limitaciones del MVP

No incluye sistema de calificaciones
No incluye chat con administrador
